Вырасти меня! Набор для выращивания «Трава для кошек»

Вырастите вкусную, сочную и полезную траву для своего кота! В составе набора семена овса. Первые росточки появятся уже через три дня, а через неделю можно предлагать лакомство для пушистого любимца. Кошки с удовольствием едят длинные травинки. Такая здоровая еда очень полезна, ведь она полна витаминов В1, В2 и В6. Кроме того, трава способствует улучшению пищеварения. Выращивайте траву прямо в пакете-упаковке. Просто достаньте семена из картонного хедера и высыпьте их в пакет, наполненный землей. Глубоко зарывать семена не нужно. Трава настолько неприхотлива, что можно смело доверить процесс выращивания детям. В комплекте: герметичный пакет, выполняющий роль горшочка для растения, органический субстрат для выращивания, семена, инструкция.
